Remember

  • pm7-ui is framework-agnostic (no React components)
  • Use simple CSS classes like pm7-button, pm7-card
  • Interactive components need data attributes (e.g., data-pm7-menu)
  • Import CSS: import '@pm7/core/dist/pm7.css'
