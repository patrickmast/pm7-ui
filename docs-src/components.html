Button

Buttons trigger actions. They can be used in forms, dialogs, toolbars, and more.

<button class="pm7-button pm7-button--ghost">Ghost</button>
<button class="pm7-button pm7-button--destructive">Destructive</button>
<button class="pm7-button pm7-button--link">Link</button>
