pm7-ui Components

Framework-agnostic UI components that work everywhere. Pure CSS styling with optional JavaScript for interactivity.

Get Started View Components

Why pm7-ui?

Framework Agnostic

Works with React, Vue, Angular, or vanilla HTML/CSS/JS. No framework lock-in.

Pure CSS Core

All styling is done with pure CSS. No build tools required for basic usage.

Accessible

Built with accessibility in mind. Keyboard navigation, ARIA attributes, and more.

Customizable

CSS custom properties for easy theming. Override any style with your own.

Tree-shakeable

Import only what you need. Each component is independently packaged.

TypeScript Support

Full TypeScript support for React and Vue packages.
