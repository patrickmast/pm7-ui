Small Size

Small size tabs are perfect for compact interfaces.

They use less vertical space while maintaining readability.

Ideal for mobile devices or dense UIs.

Medium Size (Default)

Medium size is the default and works well for most use cases.

It provides a good balance between size and readability.

Suitable for desktop and tablet interfaces.

Large Size

Large size tabs make a bold statement.

They work well as primary navigation elements.

Perfect for touch interfaces or when tabs are the main UI element.
