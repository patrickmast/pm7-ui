Card

What is a Card?

Cards are containers for displaying related content. They help organize information into digestible chunks and create visual hierarchy.

When to use

  • To group related information together
  • To create visual separation between different content sections
  • For displaying preview information that links to more details
  • In grid layouts for collections of items
  • For feature highlights or product showcases

Key features

  • Flexible structure - Header, content, and footer sections
  • Multiple variants - Default, outline, and ghost styles
  • Clickable cards - Entire card can be interactive
  • Image support - Built-in styling for card images
  • Grid layout - Easy responsive card grids
  • Composable - Mix and match card sections as needed

Anatomy

A card typically consists of:

  • Container - The card wrapper with border and shadow
  • Header - Optional title and description
  • Content - Main body of the card
  • Footer - Optional actions or metadata
  • Image - Optional visual element

CSS Classes

Class Description
.pm7-card Base card container
.pm7-card--outline Outline variant (border only)
.pm7-card--ghost Ghost variant (no border/shadow)
.pm7-card--clickable Clickable card with hover effects
.pm7-card-header Card header section
.pm7-card-title Card title
.pm7-card-description Card description/subtitle
.pm7-card-content Main card content area
.pm7-card-footer Card footer section
.pm7-card-image Full-width card image
.pm7-card-grid Grid container for cards

Best Practices

  • Consistent sizing - Use grid layouts for equal-height cards
  • Clear hierarchy - Use title, description, and content sections appropriately
  • Actionable cards - Make entire card clickable when it represents a single action
  • Spacing - Maintain consistent padding and margins
  • Images - Use appropriate aspect ratios for card images

React Usage

import { Card, CardHeader, CardTitle, CardDescription, 
         CardContent, CardFooter } from '@pm7/react';

function MyComponent() {
  return (
    <Card variant="outline">
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Card content goes here</p>
      </CardContent>
      <CardFooter>
        <Button variant="outline">Cancel</Button>
        <Button>Save</Button>
      </CardFooter>
    </Card>
  );
}
