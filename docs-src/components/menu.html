Menu / Dropdown

What is a Menu?

The PM7 Menu component provides contextual navigation and actions through a dropdown overlay. It appears when users interact with a trigger element, typically a button.

When to use

  • When you need to display a list of actions or navigation options
  • To save space by hiding secondary actions until needed
  • For context menus that appear on right-click or other interactions
  • When you have grouped actions that belong together

Key features

  • Accessible - Full keyboard navigation and screen reader support
  • Flexible positioning - Automatically adjusts to stay within viewport
  • Nested menus - Support for submenus and hierarchical navigation
  • Custom triggers - Can be triggered by any element
  • Icons support - Add icons to menu items for better visual hierarchy

Anatomy

A menu consists of:

  • Trigger - The element that opens the menu (usually a button)
  • Content - The dropdown container
  • Items - Individual menu options
  • Separators - Visual dividers between groups
  • Submenus - Nested menu structures

Installation

npm install @pm7/core

Basic Usage

<div class="pm7-menu" data-pm7-menu>
  <button class="pm7-button pm7-menu-trigger">
    Open Menu
  </button>
  <div class="pm7-menu-content">
    <button class="pm7-menu-item">Item 1</button>
    <button class="pm7-menu-item">Item 2</button>
  </div>
</div>

JavaScript Initialization

import { PM7Menu } from '@pm7/core';

// Initialize all menus
const menus = document.querySelectorAll('[data-pm7-menu]');
menus.forEach(menu => new PM7Menu(menu));

// Or initialize specific menu
const menu = new PM7Menu(document.getElementById('my-menu'));

CSS Classes

Class Description
.pm7-menu Container element for the menu
.pm7-menu-trigger Element that opens the menu (usually a button)
.pm7-menu-content Dropdown content container
.pm7-menu-item Individual menu item
.pm7-menu-item--danger Destructive action styling (red text)
.pm7-menu-separator Horizontal line to separate groups
.pm7-menu-sub Submenu content container
.pm7-menu--open Applied when menu is open

JavaScript API

Method Description
new PM7Menu(element) Initialize a menu on the given element
menu.open() Programmatically open the menu
menu.close() Programmatically close the menu
menu.toggle() Toggle menu open/closed state
menu.destroy() Remove event listeners and cleanup

Accessibility

  • Full keyboard navigation (Arrow keys, Enter, Escape)
  • ARIA attributes for screen readers
  • Focus management and trapping
  • Proper role attributes

React Usage

import { Menu } from '@pm7/react';

function MyComponent() {
  return (
    <Menu
      trigger={<button>Open Menu</button>}
      items={[
        { label: 'Profile', onClick: () => {} },
        { label: 'Settings', onClick: () => {} },
        { type: 'separator' },
        { label: 'Sign Out', onClick: () => {} }
      ]}
    />
  );
}
