Menu / Dropdown

Dropdown menus display a list of choices on temporary surfaces. They appear when users interact with a button, action, or other control.

Basic Usage

<div class="pm7-menu" data-pm7-menu>
  <button class="pm7-menu-trigger pm7-button pm7-button--primary">
    <svg width="20" height="20" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
      <line x1="3" y1="12" x2="21" y2="12"></line>
      <line x1="3" y1="6" x2="21" y2="6"></line>
      <line x1="3" y1="18" x2="21" y2="18"></line>
    </svg>
    Menu
  </button>
  <div class="pm7-menu-content">
    <button class="pm7-menu-item">Profile</button>
    <button class="pm7-menu-item">Settings</button>
    <button class="pm7-menu-item">Help & Support</button>
    <div class="pm7-menu-separator"></div>
    <button class="pm7-menu-item">Sign Out</button>
  </div>
</div>

JavaScript Initialization

Menus with the data-pm7-menu attribute are automatically initialized. For manual initialization:

<script type="module">
  import { PM7Menu } from '@pm7/core';
  
  // Manual initialization
  const menuElement = document.querySelector('.pm7-menu');
  const menu = new PM7Menu(menuElement);
  
  // Listen for menu item selection
  menuElement.addEventListener('pm7-menu-select', (e) => {
    console.log('Selected:', e.detail.item);
  });
</script>

CSS Classes

Class Description
pm7-menu Container element for the menu
pm7-menu-trigger Button that opens the menu
pm7-menu-content Dropdown container
pm7-menu-content--end Right-align the dropdown
pm7-menu-content--center Center-align the dropdown
pm7-menu-item Menu item
pm7-menu-item-icon Icon within a menu item
pm7-menu-separator Horizontal separator
pm7-menu-label Non-clickable label

Accessibility Features

  • Full keyboard navigation with arrow keys
  • Home/End keys jump to first/last item
  • Escape key closes the menu
  • Focus management and restoration
  • ARIA attributes for screen readers
  • Click outside to close

<button class="pm7-menu-item" data-active="true">
  <svg class="pm7-menu-item-icon">...</svg>
  Home
</button>

<!-- Or use inline styles for custom active state -->
<button class="pm7-menu-item" style="background-color: var(--pm7-primary); color: white;">
  Home
</button>

React Usage

When using the React package (@pm7/react):

import { Menu } from '@pm7/react';

function App() {
  const menuItems = [
    { id: '1', label: 'Profile', onClick: () => {} },
    { id: '2', label: 'Settings', onClick: () => {} },
    { id: 'sep', type: 'separator' },
    { id: '3', label: 'Sign Out', onClick: () => {} }
  ];
  
  return (
    <Menu 
      trigger={<Button>Open Menu</Button>}
      items={menuItems}
      align="start"
      onSelect={(item) => console.log('Selected:', item)}
    />
  );
}
