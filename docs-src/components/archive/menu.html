Overview

The Menu component displays a list of options on temporary surfaces. They appear when users interact with a button, action, or other control.

Key Features

  • Keyboard navigation support
  • Click outside to close
  • Flexible trigger styles
  • Support for icons and separators
  • Submenu support
  • Multiple alignment options
  • Accessible with ARIA attributes
  • No dependencies - pure CSS/JS

When to Use

  • Presenting a list of actions or options
  • Navigation menus in limited space
  • Context menus for specific elements
  • User account menus
  • Settings or preference menus

Basic Usage

<div class="pm7-menu" data-pm7-menu>
  <button class="pm7-menu-trigger pm7-button pm7-button--primary">
    Open Menu
  </button>
  <div class="pm7-menu-content">
    <button class="pm7-menu-item">Profile</button>
    <button class="pm7-menu-item">Settings</button>
    <div class="pm7-menu-separator"></div>
    <button class="pm7-menu-item">Sign Out</button>
  </div>
</div>

JavaScript Initialization

Menus with data-pm7-menu are automatically initialized. For manual control:

import { PM7Menu } from '@pm7/core';

// Manual initialization
const menuElement = document.querySelector('.pm7-menu');
const menu = new PM7Menu(menuElement);

// Listen for selection
menuElement.addEventListener('pm7-menu-select', (e) => {
  console.log('Selected:', e.detail.item);
});

Alignment Options

<!-- Left aligned (default) -->
<div class="pm7-menu-content">...</div>

<!-- Right aligned -->
<div class="pm7-menu-content pm7-menu-content--end">...</div>

<!-- Center aligned -->
<div class="pm7-menu-content pm7-menu-content--center">...</div>

CSS Classes

Class Description
pm7-menu Container element for the menu
pm7-menu-trigger Button that opens the menu
pm7-menu-content Dropdown container
pm7-menu-content--end Right-align the dropdown
pm7-menu-content--center Center-align the dropdown
pm7-menu-item Menu item
pm7-menu-item-icon Icon within a menu item
pm7-menu-separator Horizontal separator
pm7-menu-label Small uppercase label
pm7-menu-section-title Large section heading
pm7-menu-item--has-submenu Item with submenu indicator

Keyboard Navigation

  • Tab - Move focus to menu trigger
  • Enter / Space - Open menu when trigger focused
  • ↓ - Move to next item
  • ↑ - Move to previous item
  • Home - Jump to first item
  • End - Jump to last item
  • Escape - Close menu
  • Enter / Space - Select focused item

JavaScript API

Method Description
open() Opens the menu
close() Closes the menu
toggle() Toggles the menu open/closed state

Events

Event Description Detail
pm7-menu-select Fired when a menu item is selected { item: HTMLElement, menu: PM7Menu }
