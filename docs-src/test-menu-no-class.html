AI-Agent FIRST: Menu Test

This test proves that menus work with ONLY the data-pm7-menu attribute.

Test 1: Menu WITHOUT class="pm7-menu"

This menu has NO CSS class, only the data attribute:

<div data-pm7-menu>
  <button class="pm7-menu-trigger">Click Me</button>
  <div class="pm7-menu-content">...</div>
</div>

How it works

The JavaScript automatically adds the required pm7-menu class when it finds an element with data-pm7-menu.

This is the essence of AI-Agent FIRST design:

  • ONE attribute = fully working component
  • No confusion about which classes are needed
  • JavaScript handles the setup automatically
