Fixed Position Theme Switches

Fixed position theme switches stay in viewport corners and are perfect for applications where theme switching should always be accessible.

Fixed Icon (Bottom Right)

This demo shows a fixed icon-only theme switch in the bottom right corner.

It's circular, minimal, and doesn't interfere with content.

In real usage, this would be positioned fixed to the viewport.

Fixed with Label (Bottom Right)

This variant includes a label and is positioned in the bottom right.

Great for applications where clarity is important.

Theme

The label helps users understand the toggle's purpose.

Different Sizes

Fixed position theme switches can use different sizes.

Use small size for minimal UI, large for better accessibility.
