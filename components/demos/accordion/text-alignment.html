Text Alignment

Control the alignment of text in accordion headers

Left Aligned (Default)

This is the default alignment. Text is aligned to the left side of the accordion header.

No additional classes needed for left alignment as it's the default.

Center Aligned

This accordion has center-aligned text in the header using the pm7-accordion--text-center class.

Perfect for creating a symmetrical, balanced appearance in your accordion headers.

Right Aligned

This accordion has right-aligned text in the header using the pm7-accordion--text-right class.

Useful for right-to-left languages or specific design requirements.

Combined: Icon Start + Center Text

This combines icon at start with center-aligned text. Classes can be combined for different effects.

You can combine different modifiers to achieve the exact layout you need.
