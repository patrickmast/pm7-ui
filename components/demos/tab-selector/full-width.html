Full Width Tabs

Full width tabs stretch to fill their container.

Each tab gets an equal share of the available space.

This layout works well for mobile interfaces.

It ensures tabs are always easy to tap.

Combine with solid or pills variant for best effect.

The full width modifier works with all variants.

Full Width Pills

Pills variant with full width layout.

Great for settings or configuration screens.

Works with any number of tabs.

Tabs automatically adjust to available space.
